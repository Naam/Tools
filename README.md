Tools
=====

Some quick tools I made to simplify my tasks

# XOR
<code>./xor < key > \[inputFile\] \[outputFile\]</code>  
Here is some usage example:  
<code>./xor NotSoStrongKey secretImage.png newimage.png</code>  
But you can also use pipe or redirections :  
<code>cat textfile | ./xor AnotherBadKey > newFile.ext</code>
